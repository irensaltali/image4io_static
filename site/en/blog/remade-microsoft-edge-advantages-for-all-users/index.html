Remade Microsoft Edge: Advantages for all users

25 June 2019, Web Developer

Last week, Microsoft announced preliminary versions of its remade Edge browser for Windows 10, as well as for Windows 7, Windows 8, Windows 8.1 and macOS.

Last year, Microsoft announced that it would give up its own browser technologies and use Chromium open source project instead. The company replaced Edge's internal rendering engine called "EdgeHTML" with "Blink", which already runs Google Chrome. Joe Belfiore, Corporate Vice President in the Experiences and Devices division at Microsoft explains the main point of this decision:

“… we intend to adopt the Chromium open source project in the development of Microsoft Edge on the desktop to create better web compatibility for our customers and less fragmentation of the web for all web developers.”

Advantages for Web Developers

Besides all the advantages designed for end users, the outstanding feature of this remade Edge browser is the alignment with many other web standards, mainly Chromium-based browsers. Microsoft Edge targets delivery of higher compatibility for everyone and offering much simpler test-matrix for developers. The company also commits to keeping Microsoft Edge updated for all supported Windows versions and scheduling these updates on a more frequent basis.

New Features for End Users

Microsoft Edge has brought many new features designed for better end user experience: such as focus improvement for reading, both online and offline dictionary option while reading, organization tools for tabs, so on. Microsoft Edge is now offering easy to change permission tools, removal of pop ups, private browsing. In general, Microsoft offers end users easy organization, privacy and focus tools with customization.

Compatibility and Web Performance

In short, the newcomer Microsoft Edge is based on Google’s open-source Chromium rendering engine, which also powers other popular browsers like Chrome, Opera, and Vivaldi, among many others.
